# ft_printf_mericodes
ft_printf is a basic implementation of the printf function.
is has:
  - %c to print chars
  - %s to print strings
  - %p to print a pointer
  - %d to print decimals
  - %i to print integers
  - %u to print unsigned ints
  - %x to print hexadecimals in lowercase
  - %X to print hexadecimals in uppercase
  - %% to print a '%'
